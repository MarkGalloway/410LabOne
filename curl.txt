        <div class="post-content drop">
          <p>I am not big on resolutions, but like many of you, I’d sure like to write more this year. Writing was the single biggest factor in helping my career, and it’s led me to start new projects, find new interests, helped solved problems, etc. I’m a horrible note-taker, and far too many thoughts and ideas permenantly live in my mind. Which doesn’t scale—especially for someone at my age.</p>

<p>So I’m writing this now. I’m writing it in a plain ol’ text editor using Markdown. I’ll save this file and with a quick command line, it’ll sync with my code on GitHub and be published to my webserver as a <a href="http://jekyllrb.com">static HTML</a> file. I’m going back to basics. Hypertext.</p>

<blockquote>
  <p>“Getting back to basics feels good … Just me and as many blank text files as I can fill up.”</p>
</blockquote>

<p>And getting back to basics feels good. Less to get in the way. I no longer care if I’m doing things the best possible way, I just care about writing and publishing. Over the last few years I’ve been busy building a <a href="https://dribbble.com">business</a> and rebuilding a life. I’d love to share more of that here.</p>
